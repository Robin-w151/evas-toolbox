# Evas Toolbox

## Developer

### Install dependencies

```shell
pnpm install
```

### Run DEV server

```shell
pnpm dev
```

### Run PROD build

```shell
pnpm build
```

### Run prettier and eslint

```shell
pnpm lint
```

### Run svelte checks

```shell
pnpm check
```

### Run playwright tests

```shell
npx playwright install
pnpm test
```
